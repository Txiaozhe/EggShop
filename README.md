# EggShop

一个用 eggjs 框架开发的后台商城系统。

## 获取和运行
```shell
$ git clone https://github.com/Txiaozhe/EggShop.git
$ cd EggShop
$ npm install
$ npm run dev    # 开发调试阶段
$ npm run build  # 部署阶段
```
### 运行环境
* Windows/OSX/Linux  NodeJS
* Docker
